Predictive Maintenance of a Lathe

App to predict the overstrain of the tool used in a lathe in function of the torque [Nm] and the tool wear[min], and to predict the class of the process.

Prediction

Overstrain prediction

The range of the torque is [-8.39 ; 2.48], and the one of the tool wear is [-4.03 ; 1.22]. Those entries are standardized

Class of the process

The entry for type will be as followed: High = 0; Low = 1; Medium = 2.
Class 0 = Lathe okay
Class 1 = Lathe failed
Dif temperature is the diference between the process temperature and air temperature in Kelvin.
